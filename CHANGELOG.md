# Changelog

## v1.0.0

Первый стабильный релиз spelunk-md.

### Возможности
- Генерация файлов контекста для ИИ: `CLAUDE.md`, `AGENTS.md`, `.cursorrules`, `.windsurfrules`.
- Автоматическое сканирование структуры проекта с учётом `.gitignore`, включая вложенные конфиги.
- Определение стека технологий по расширениям файлов и манифестам (`go.mod`, `package.json`, `Cargo.toml` и другие).
- Извлечение сигнатур публичных функций, типов, констант и переменных.
- Чтение git-истории и метаданных проекта (лицензия, авторы, CI, зависимости).
- Инкрементальное обновление контекстных файлов на основе изменений в git.
- Хранение API-ключа в системном keyring с fallback на переменную окружения `OPENROUTER_API_KEY`.
- Команда `custom` для генерации в произвольный файл.
- Поддержка `--dry-run`, `--model`, `--timeout`, `--force` и других флагов.
- Публикация бинарников для Linux, macOS и Windows через GoReleaser.
